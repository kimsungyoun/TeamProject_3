# 개발현황
1. 게시판 기능 구현 :  공지, 자료공유, 커뮤니티
2. 관리자 페이지 구현 (학생 정보 수정, 신고내역 확인)
3. 게시글 및 댓글 신고 기능 구현

# 느낀 점
1. 깃 허브를 통한 첫 협업을 하는데 있어서 깃 허브의 복잡함과 불편함을 느꼈음(누군가 바꿔버린 본인 코드로 인한 충돌현상)
